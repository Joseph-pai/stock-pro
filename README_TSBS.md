# 台股爆發預警系統 (TSBS)

一個基於 Next.js 14 的輕量化 Web App，專為監測台股爆發性成長標的設計。

## 核心功能
- **爆發評分**: 整合量能、均線、法人籌碼進行即時評分。
- **PWA 支持**: 可安裝於 iOS/Android 桌面。
- **自適應視圖**: 
  - 豎屏: 爆發名單列表與評分。
  - 橫屏: 自動切換為全螢幕 K 線技術分析圖表。

## 如何運行
1. 確保已安裝 Node.js。
2. 進入專案目錄: `cd tsbs-app`
3. 安裝依賴: `npm install`
4. 啟動開發伺服器: `npm run dev`
5. 打開瀏覽器訪問: `http://localhost:3000`

## 技術指標說明
- **量能比 (V-Ratio)**: 當日成交量 / 過去 20 日平均成交量。
- **均線糾結**: MA5, MA10, MA20 在 3% 區間內擠壓。
- **突破訊號**: 價格站上所有短期均線並伴隨量能爆發。
- **投信籌碼**: 核心權重，監測投信是否連續 3 日買超。
